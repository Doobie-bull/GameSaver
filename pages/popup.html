F12 will bring up the devtools, where you can find the "Game Saver" tab to see the list of files the extension has captured.

If you have any issues feel free to report them at GitHub
If you cannot download your games due to a bug or issue, I can help you recover the files, just post an issue on github.

This project is not directly affiliated with Bluemaxima's Flashpoint, although I am active on discord. You can find me there for more direct help (krum110487). You can upload the files however you want, but I suggest to use Internet Archive or to Flashpoint's Flashfreeze (or both!)
